# VERSEAL — E-commerce Artístico

Bem-vindo ao **Verseal**, um e-commerce artístico desenvolvido com **PHP, HTML, CSS e JavaScript**, com foco em responsividade e identidade visual própria.  
O projeto está em constante evolução e serve como base para construção de uma plataforma completa de venda de produtos artísticos e gerenciamento de usuários.

---

## Visão Geral do Projeto

O Verseal oferece uma estrutura de site com:

- Página inicial com artes e destaque visual  
- Sistema de login e sessão  
- Perfis de usuários e artistas  
- Upload de imagens  
- Estrutura organizada de backend  
- Frontend estilizado e responsivo  

Este repositório representa o código-fonte completo do site em desenvolvimento.

---

## Tecnologias Utilizadas

- **PHP** — backend e controle de sessões  
- **MySQL** — banco de dados  
- **HTML5** — estrutura  
- **CSS3** — estilização personalizada  
- **JavaScript** — interações e animações  
- Hospedagem local utilizando **XAMPP / WAMP / Laragon**

## Como Rodar o Projeto

1. **Clone o repositório:**
   
   git clone https://github.com/pedrozxxz/verseal.git

**Coloque o projeto dentro do htdocs (XAMPP) ou www (WAMP):**

2. **Copiar código**
   
C:/xampp/htdocs/verseal

**Configure o banco no arquivo:**

config.php

**Inicie o servidor local**

Abra o navegador e acesse:

http://localhost/verseal
